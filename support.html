Support

Solo-developer app — you get a direct line.

Get help

Email support@iantrimble.com. Most messages get a response within 24 hours on weekdays. Please include:

  • Your macOS version (Apple menu → About This Mac)
  • Lectern version (visible in Settings or Help → About Lectern AI)
  • A one-line description of what you were doing
  • A screenshot if anything looks visually wrong
  • The LLM provider you were using when the issue happened

Report a bug

For reproducible bugs, tell me the exact steps. If the app crashed, macOS may have recorded a crash log at ~/Library/Logs/DiagnosticReports/; attaching that file helps a lot.

Request a feature

Feature requests are welcome. The app's roadmap is solo-developer pace — I'll tell you honestly whether something is on the near-term list, the long-term list, or out of scope.

Press & review copies

Journalists and reviewers — email press@iantrimble.com and I'll send a promo code for a free year of Pro plus the press kit. No paywall on honest reviews.

What we don't support

I can't help debug API keys you obtained from an LLM provider — use that provider's support. I can only tell you whether Lectern's auth path is behaving correctly. If a specific deck's output is bad, the first thing to try is changing your LLM provider — Lectern supports seven, and quality varies significantly by deck topic.
